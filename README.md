# Portfolio Website

This is a personal portfolio website for Muhammad Atif Latif, AI Engineer & Data Scientist.

## How to View

Simply open `index.html` in your web browser.

## Structure

- `index.html`: Main HTML file.
- `style.css`: Stylesheet with dark theme and glassmorphism effects.
- `script.js`: JavaScript for smooth scrolling and animations.
- `Profile.pdf`: The resume file (linked in the Hero section).

## Customization

- **Images**: Replace the placeholder in the hero section with your actual photo.
- **Links**: Update the social media links in `index.html`.
- **Projects**: Add more projects to the projects section.
